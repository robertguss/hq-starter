## An LLM Framework

for LLMs

A lightweight orchestrator that lets AI build and maintain workflows.

[Start Building](https://github.com/The-Pocket-World/Pocket-Flow-Framework)

## Core Concepts

### Single-Step Nodes

Encapsulate individual LLM calls, API requests, or data transformations. Easy to debug and reuse.

### Multi-Step Nodes

Bundle multiple sub-tasks into one atomic action. Perfect for chaining LLM prompts or complex transformations.

### Sub-Flows

Nest entire flows inside nodes for granular control while keeping top-level flows clean and manageable.

### AI-First Design

Built specifically for LLMs, RAG, and multi-agent systems. Handles failures gracefully and scales effortlessly.

### Modular & Composable

Create reusable node templates for common tasks like document summarization or embedding generation.

### Enterprise-Ready

Stable orchestration layer with robust retry logic and partial failure handling built-in.

### Developer-Friendly

Minimalist codebase keeps friction low while supporting advanced features like batch processing and async execution.

In the future, LLM projects will be self-programmed by LLMs themselves.
