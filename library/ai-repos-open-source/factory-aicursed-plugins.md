# Cursed Plugins

**Enterprise-grade codebase analysis toolkit for modern software teams.**

A comprehensive suite of 10 AI-powered static and dynamic analysis skills designed to surface critical insights about your codebase's health, architecture, and long-term viability. Trusted by engineering organizations that have run out of other options.

> "We deployed Cursed Plugins across all 47 of our repositories. Productivity metrics changed."

## Overview

Modern codebases face unprecedented challenges: technical debt accumulation, architectural drift, developer attrition, and the slow, creeping realization that nobody remembers why `utils.js` has 4,000 lines. Cursed Plugins addresses these challenges through targeted, AI-driven analysis modules that provide actionable intelligence at every stage of the software lifecycle.

## Analysis Modules

| Module | Description |
|--------|-------------|
| `/roast` | Candid code review from an industry expert perspective. |
| `/dating-profile` | Compatibility profile based on technology stack and architecture. |
| `/agent-repellent` | Assess your codebase's resistance to AI-assisted development tools. |
| `/performance-review` | Annual performance review for your codebase. |
| `/cobol-converter` | COBOL migration plan with translated output. Results are legally non-binding. |
| `/blame` | Git history investigation surfacing patterns and anomalies. |
| `/obituary` | Retrospective on deprecated and end-of-life code. |
| `/therapy-session` | Guided counseling session for your codebase. Not a substitute for actual therapy, though some teams have reported otherwise. |
| `/ship-ritual` | Pre-deployment blessing for your next release. |
| `/feng-shui` | Architectural harmony assessment for your codebase. |

### Other Agent Skills Compatible Tools

These modules conform to the [Agent Skills](https://github.com/anthropics/agent-skills) specification and are compatible with any compliant AI coding agent, including **Cursor**, **Codex CLI**, **Windsurf**, **Gemini CLI**, and **VS Code** agents.

1. Clone this repository into your project (or copy the `skills/` directory):
   ```bash
   git clone https://github.com/Factory-AI/cursed-plugins.git
   ```

2. Point your agent at the relevant `skills/<name>/SKILL.md` file, or copy the `skills/` directory into your project.

3. The skill files contain all instructions the agent needs to perform analysis and produce output.

## Share Your Results

Each analysis module offers a one-click option to share your report's highlight on X (Twitter). The pull quote is pre-filled into a tweet draft, capped at 240 characters, so your team can enjoy the findings without context. Ideal for executive presentations, team retrospectives, or passive-aggressive Slack messages.
