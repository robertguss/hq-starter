## The best SQLite database editor for macOS

Base is a small, powerful, comfortable SQLite editor for everyone

## Detailed Schema Inspector

Manage your database structure with Base's schema inspector, making understanding your tables effortless.

View table schemas, column types, constraints, and relationships at a glance. Understand your table design without digging through SQL.

![Image 3: Schema Inspector](https://menial.co.uk/img/base3-schema@2x.png)

## Intuitive Table Editor

Create and modify tables with ease using Base's visual table editor. No need to write complex `CREATE` or `ALTER` statements.

Add and organize columns and define constraints through a clean interface. The table editor makes database design accessible to everyone.

![Image 4: Table Editor](https://menial.co.uk/img/base3-table-editor@1x.png)

## Simple Data Browser

Browse, filter and edit your table contents with ease.

View your data in a simple table format. You can edit cells directly and use powerful filtering tools to find exactly what you need. The data browser makes working with your database contents easy and efficient.

![Image 5: Data browser](https://menial.co.uk/img/base3-data-browser@1x.png)

## SQL Editor & Query Tools

Write and execute SQL queries with intelligent autocomplete and syntax highlighting.

Store your common queries as reusable snippets, view query results instantly, and take advantage of helpful features that make SQL more accessible. Perfect for both learning SQL and running complex queries efficiently.

![Image 6: SQL Editor](https://menial.co.uk/img/base3-dark-sql@1x.png)

## Import & Export

Import data from CSV and SQL dump files. Export your results to SQL, delimited text, JSON, and Excel formats.

Move your data in and out of Base quickly and easily - whether you're migrating from another system or sharing results with colleagues.
