Get started

## One command. That's it.

LazyPi installs Pi if you don't have it, then sets everything up. Re-run anytime — it skips what's already installed.

npx @robzolkos/lazypi

[Read First Steps](/docs/first-steps.html) [Installation details](/docs/installation.html)

Installed? Start with [First Steps](/docs/first-steps.html) to launch Pi, authenticate, and try a few high-value commands.
