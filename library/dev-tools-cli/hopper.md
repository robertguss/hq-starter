# Hopper

## The fast and native macOS disassembler

[Try It Free](https://www.hopperapp.com/download.html)[Buy Now](https://www.hopperapp.com/webstore)

### Everything You Need for Reverse Engineering

Hopper combines powerful analysis capabilities with an intuitive Mac-native interface, giving you professional-grade tools at an affordable price.

![Image 2: Hopper v4 Main Screen](https://www.hopperapp.com/Screenshots/hopv6-main@1x.webp)
Hopper Disassembler for Mac requires macOS 12.4 or higher.

# Native UI

Built with a blazingly fast, beautifully polished Cocoa interface that prioritizes keyboard shortcuts and native macOS integration for a smooth user experience.

# AI

Now enhanced with AI capabilities through an integrated MCP server, enabling seamless connections to your preferred large language models for intelligent analysis.

# Control Flow Graph

Interactive CFG visualization with PDF export. Navigate complex code structures with ease and annotate directly from the graph view.

# Decompiler

Transform assembly code into readable C-like pseudo-code. Advanced analysis separates code from data and identifies function parameters automatically.

Hopper displays the code using different representations. Take a look at the assembly, the CFG, and the pseudo-code of a procedure, all at the same time.

![Image 3](https://www.hopperapp.com/Screenshots/hopv6-modes@1x.webp)

# Objective-C & Swift

Specialized for macOS and iOS development. Decode mangled Swift names and extract Objective-C selectors and messages.

# Debugger

Dynamic analysis with LLDB and GDB support. Debug your Mac applications while analyzing their structure in real-time.

![Image 4](https://www.hopperapp.com/Screenshots/hopv6-cfg-edit@1x.webp)
Edit, annotate, comment your work directly from the interactive CFG view.

# Scriptable

Automate analysis with powerful Python scripting support. Access most Hopper features programmatically for custom workflows.

# Extensible SDK

Extend Hopper's capabilities by adding custom file format parsers and CPU architecture backends to support your specific reverse engineering requirements and workflows.

Find what you are looking for. The inspector panel automatically adapts its content according to the context.
