Self-managed nodes require a commercial license and license key. Self-managed Akka services have Akka clustering-from-within for scaling and resilience. You can package Akka services as standalone binaries, Docker images, or Akka images. Deploy and run Akka services into any container runtime including Heroku, Linode, Render, and Fly.io. Or, deploy and run Akka into your own environments whether powered by bare metal, VMs or Kubernetes. You are responsible for separately managing routes, deployment, persistence, and security.
