Retype turns your Markdown files into a beautiful, production-ready website. No design work, no coding, no recurring fees, **and no vendor lock-in**. Write your content, run one command, and your site is ready to share.

Host for free on [GitHub Pages](https://retype.com/hosting/github-pages/), [Netlify](https://retype.com/hosting/netlify/), or [Cloudflare](https://retype.com/hosting/cloudflare/), or deploy to any web server you choose. Your content lives in plain Markdown files that you own and control, with no proprietary formats and no surprises.

All you need is a single `.md` file.

## How it works

[1](https://retype.com/#install-retype)

Install Retype

You can install Retype using [npm](https://www.npmjs.com/get-npm), [Yarn](https://classic.yarnpkg.com/en/docs/install/), or the [dotnet](https://dotnet.microsoft.com/download/dotnet-core) CLI.

[NPM](https://retype.com/#npm)

[Yarn](https://retype.com/#yarn)

[dotnet](https://retype.com/#dotnet)

```none
npm install retypeapp --global
```

[2](https://retype.com/#start-your-project)

Start your project

Navigate to any folder with Markdown files and run `retype start`.

```none
retype start
```

Retype builds your site and opens it in the browser automatically.

## Why Retype?

Retype is designed to get out of your way so you can focus on writing. There’s no complex build pipeline, no framework to learn, and no dependencies to maintain.

### It just works

Retype has been built to be easy to use and should **_just work_** out-of-the-box without any special configuration, technical knowledge, or troublesome setup.

### Lightning fast

Don't blink. Retype was built for speed.

### Easy install

[Installation](https://retype.com/guides/getting-started/) takes only a few seconds. Then all you need is one Markdown `.md` file which Retype will start building a new website from.

### Powerful

Project level [configuration](https://retype.com/configuration/project/) using **retype.yml** unlocks many more features and customization.

### Simple formatting

Pages are formatted using [Markdown](https://retype.com/guides/markdown/) syntax and Retype [components](https://retype.com/components/). Page level [configuration](https://retype.com/configuration/page/) is available, such as setting a custom navigation [`label`](https://retype.com/configuration/page/#label) or [`icon`](https://retype.com/configuration/page/#icon).

### Live reload

If a change is detected, such as editing and saving an `.md` file, your Retype website will be updated almost instantly within the browser.

### Host anywhere

Retype generates an HTML website that you can self-host on any web hosting service, or for free using [GitHub Pages](https://retype.com/hosting/github-pages/), [Netlify](https://retype.com/hosting/netlify/), or [Cloudflare](https://retype.com/hosting/cloudflare/). No special server-side software or external dependencies are required. You can host your Retype site as a public website or as a private website within your organization's network.

> “Retype is the perfect fit for my user persona - tech savvy non-devs who want to write using the best web standards out there.”
