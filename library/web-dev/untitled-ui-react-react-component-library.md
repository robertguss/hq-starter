### Does Untitled UI React include an MCP?

Yes, Untitled UI provides our own MCP (Model Context Protocol) integration for [Untitled UI React](https://www.untitledui.com/react), so AI assistants can access the component library directly from your development environment.

With the Untitled UI MCP server, tools like Claude Code, Cursor, Gemini CLI, and other MCP-compatible AI assistants can browse, search, and install components using natural language. Instead of manually copying and pasting code, your AI assistant can discover components, generate layouts, and run the correct CLI commands to add them to your project automatically.

Once connected, you can ask your AI assistant to do things like add components, build page layouts, or search the library for specific UI patterns, making it faster to prototype and build interfaces directly from prompts.

Check out our [installation guide](https://www.untitledui.com/react/integrations/mcp) to learn more.

Tech stack

### What tech stack does Untitled UI React use?

Untitled UI React components are built with and powered by:

*   [React v19.2](https://react.dev/)
*   [Tailwind CSS v4.2](https://tailwindcss.com/blog)
*   [TypeScript v5.9](https://www.typescriptlang.org/docs/handbook/release-notes/typescript-5-9.html)
*   [React Aria](https://react-spectrum.adobe.com/react-aria/index.html)

We’ve kept our tech stack minimal, so you can focus on building instead of wrestling with dependencies. You own the code so you can build and extend without relying on third-party maintainers.

We're always working to make Untitled UI better than ever, including implementing new updates into our products as fast as possible. All Untitled UI products come with free updates for life.

### What version of Tailwind CSS does Untitled UI React use?

Untitled UI React components are built using the latest version of [Tailwind CSS (v4.2)](https://tailwindcss.com/blog), giving you access to the newest utility classes, performance improvements, and best practices from the Tailwind team.

### Do Untitled UI React components work with React v19.2?

Untitled UI React components are built with the latest version of [React (v19.2)](https://react.dev/blog/2025/10/01/react-19-2), so you get modern features, improved performance, and full compatibility with the newest React updates out of the box.

### What version of TypeScript does Untitled UI React use?

Untitled UI React components are built with the latest version of [TypeScript (v5.9)](https://www.typescriptlang.org/docs/handbook/release-notes/typescript-5-9.html), so you get modern features, better type safety, and performance improvements out of the box.

### What are Untitled UI React components based on?

To ensure every component meets modern accessibility standards and best practices, Untitled UI React components are built on top of [React Aria](https://react-spectrum.adobe.com/react-aria/index.html).

React Aria is a library of unstyled primitives that make it easy to build high-quality and accessible components. React Aria is actively maintained and continuously improved by Adobe, ensuring long-term reliability and support.

### What browsers are supported?

All Untitled UI React components are built to work seamlessly in the latest stable versions of major browsers, including Chrome, Firefox, Safari, and Edge.

Internet Explorer 11 is not supported.

### Can I use Untitled UI React with Vue, Svelte, or other JS frameworks?

Untitled UI React components are built specifically for React. However, the Tailwind tokens are **framework-agnostic** and can be used in almost any project that supports Tailwind CSS!

To integrate them, simply run `npx @untitleduico/cli tailwind` in your project directory to scaffold a new Untitled UI project.

Check out our [installation guide](https://www.untitledui.com/react/docs/installation) to learn more.
