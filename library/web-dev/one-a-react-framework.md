The simplest, fastest, all-in-one React Native framework.

A single Vite plugin for fully-typed[file-system routes](https://onestack.dev/docs/routing), per-page[render modes](https://onestack.dev/docs/render-modes),[loaders](https://onestack.dev/docs/routing-loader),[middleware](https://onestack.dev/docs/routing-middlewares), a[ton of features](https://onestack.dev/docs/features), and a production-ready[Hono](https://hono.dev/), Vercel or Cloudflare server.

Get started:

npx one

5m video intro

##### Typed FS Routing

Typed file-system routing, nested layouts with groups.

##### Render Modes

Render any page as SPA, SSR, or SSG, control the global default.

##### Loaders

Typed loaders make it easy to bring in data and migrate from other frameworks.

##### Web + Native

Build a website with React. Or a native app with React Native. Or both at once.

##### Vite-native

Use just Vite, even for native, or defer to Metro. One lets you choose.

##### Simplistic model

One avoids the complexity of RSC in favor of SSG, SPA, or SSR, with loaders or a sync-engine for data.
