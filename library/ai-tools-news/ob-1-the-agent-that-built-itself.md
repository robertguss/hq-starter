## **OB-1** Owner's Manual

The multi-model coding agent for your terminal.

## Why OB1?

OB1 is a terminal-based coding agent—like Claude Code, but with multi-model access and self-optimizing workflows. You bring your own API keys (or use ours), pick the best model for each task, and let OB1 handle the rest.

**The short version:**if you're locked into a single model provider, hitting context limits, or manually repeating the same workflows, OB1 fixes that.

01 Multi-Model Claude, GPT, Gemini, Grok + 300 more. `/model` to switch. No vendor lock-in.

02 Self-Optimizing Learns from your sessions. Run `/inbox` for suggested skills and workflow improvements.

03 Session Forking Fork any conversation to explore alternatives. Rewind to any point. Share via link.

04 CC Compatible Migrate from Claude Code with `/migrate`. AGENTS.md, skills, and commands carry over.
