## Week by Week

1

### Discover Your Build

Shift from consumer to builder mindset. Use the Problem Mining framework and Build Compass to find an idea worth building, then set up your dev environment.

Deliverable: One-page Build Brief + working development environment.

2

### Visual Automation Foundations

Learn n8n from scratch — triggers, actions, logic, and data flow. Build and deploy your first working automations.

Projects: Personal news briefing + email triage assistant.

3

### Master AI Communication

Master the ICE Framework (Identity, Context, Examples) and four prompt patterns. Set up Cursor with MCP integrations for supercharged development.

Key insight: The quality of your prompts determines the quality of your builds.

4

### Build Your First Apps

Connect frontends to backends. Build webhooks, simple web interfaces, and your first end-to-end functioning app.

Projects: Form handler with AI response + personal project app.

5

### Building with Cursor

Master Cursor's Agent mode and the Agentic Coding Protocol. Build multi-component React apps and deploy to Vercel.

Project: SlideForge — a markdown-to-slides presentation app.

6

### Expanding with Claude Code

Quick builds, Chrome extensions, and deployment. Learn when to use Claude Code vs Cursor, and take your project from local to live.

Outcome: Working Chrome extension + main project deployed live.

7

### Productize Your Build

Turn your project into a product. Build a landing page, set up payment with Stripe, and create a compelling offer.

Deliverable: Live landing page with payment system configured.

8

### Launch and Beyond

Ship publicly, collect feedback, and create your 90-day Builder plan. Launch across your network, communities, and platforms.

Goal: Public launch + external validation (sale, user, or testimonial).
