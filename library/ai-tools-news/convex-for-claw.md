## Empowering the Claw Ecosystem

The[OpenClaw](https://openclaw.ai/)ecosystem is growing fast. Developers are building skills, dashboards, multi-agent systems, and full products on top of it. Every one of those projects needs a backend that can keep up with real-time agent communication, persistent state, and type-safe data.

That's where Convex comes in.

We're backing developers and founders building with OpenClaw. If your project runs on Convex (or plans to), you can apply for credits, support and swag.

## FAQs

Can I apply if I'm just building a skill for ClawHub?

Yes. If your skill project uses Convex for any backend functionality (storage, auth, API routes, vector search), you're eligible.

What if I'm building on OpenClaw but not using Convex yet?

Apply anyway. We can help you evaluate whether Convex fits your architecture, and if it does, we'll get you set up with credits.

Are funded companies eligible?

If you’ve raised funding and have more than 5 employees, we recommend the[Convex for Startups](https://convex.dev/startups)program. Smaller funded teams building in the Claw ecosystem are welcome here.

Do I need to be open source?

No. Both open source and closed source projects qualify. Open source projects may also be eligible for the[Convex for Open Source](https://convex.dev/open-source-program)program.

Can I use both this program and the Startups or OSS program?

No. Each project qualifies for one program. If you're not sure which fits best, apply here and we'll help route you.

When will I hear back?

We review applications on a rolling basis and respond within 30 days.

## Empowering the OpenClaw Ecosystem

This program is for developers and founders building tools, apps, or infrastructure in the OpenClaw ecosystem with Convex as their backend. You don't need to be a startup. You don't need funding.

Complete a short form. We review applications continuously on a rolling 30-day basis.

![Image 9](https://www.convex.dev/_next/image?url=%2Fstartups%2Feligibility-tunnel.svg&w=1024&q=75)

## You should apply if your project:

*   Uses Convex (or is migrating to it) as the primary backend
*   Is built on or integrates with OpenClaw, ClawHub, or the broader Claw ecosystem
*   Ships skills, dashboards, multi-agent systems, or developer tools for Claw users
*   **Is actively maintained with a public repo or live product**

Open source projects, indie developers, and early-stage startups all qualify. If you've raised significant funding, check out the[Convex for Startups](https://convex.dev/startups)program instead.
