# GenAI Ecosystem

![Image 2: genai art diagram 1](https://dist.neo4j.com/wp-content/uploads/20231030151119/genai-art-diagram-1.svg)

Knowledge graphs bring more accurate responses, rich context, and explainability to each generative AI model interaction. By combining knowledge graphs with native vector search, you get the best of both worlds with Graph RAG (Retrieval Augmented Generation).

Language models also can be used to extract entities and their relationships from unstructured text to build up and enrich knowledge graphs.

## [](https://neo4j.com/labs/genai-ecosystem/#_how_to_get_started)How to get started

1.   For an high level overview, have a look at [neo4j.com/genai](https://neo4j.com/generativeai/)

2.   Use the [LLM-Knowledge Graph Builder](https://neo4j.com/labs/genai-ecosystem/llm-graph-builder/) to turn your own documents into a knowledge graph

3.   Integrate one of the [Neo4j MCP servers](https://neo4j.com/developer/genai-ecosystem/model-context-protocol-mcp/#neo4j) into your AI assistant or development workflow.

4.   If you want to learn more take one of the [GenAI GraphAcademy courses](https://graphacademy.neo4j.com/categories/llms/?ref=genai-docs)

5.   Pick your [GenAI framework of choice](https://neo4j.com/labs/genai-ecosystem/genai-frameworks/) and start building your own GenAI applications with Neo4j

## [](https://neo4j.com/labs/genai-ecosystem/#_graphacademy_courses)GraphAcademy Courses

If you want to learn how LLMs and Knowledge Graphs combine to improve GenAI applications, check out the [Neo4j & LLM courses on GraphAcademy](https://graphacademy.neo4j.com/categories/llms/?ref=genai-docs).

[![Image 3: llm fundamentals](https://cdn.graphacademy.neo4j.com/assets/img/courses/banners/llm-fundamentals.png)](https://graphacademy.neo4j.com/categories/llms/?ref=genai-docs)

## [](https://neo4j.com/labs/genai-ecosystem/#_genai_ecosystem)GenAI Ecosystem

The Neo4j GenAI Ecosystem is a collection of tools and integrations that make it easy to use LLMs with Neo4j.

### [](https://neo4j.com/labs/genai-ecosystem/#_graphrag)GraphRAG

GraphRAG combines an ingestion process that extracts entities and relationships from unstructured text and further uses graph algorithms for enrichment and summarization. The retrieval step then uses the knowledge graph in combination with vector search to navigate to more relevant information than just the initial text chunks.
