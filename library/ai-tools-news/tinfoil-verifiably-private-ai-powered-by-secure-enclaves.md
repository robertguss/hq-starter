The secure enclave generates a hardware-signed attestation report using AMD. The report is cryptographically verified against AMD's root of trust, proving the code runs inside a genuine secure enclave.

Compare Measurements

The code measurement from the signed release is compared bit-for-bit against the enclave's self-reported measurement. A match proves the enclave is running exactly the expected, unmodified code committed to Sigstore.
