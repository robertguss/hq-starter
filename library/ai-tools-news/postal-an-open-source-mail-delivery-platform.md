# ![Image 3](https://docs.postalserver.io/logo-dark.svg)

**Postal** is a complete and fully featured mail delivery platform for use by websites & web servers. Think Sendgrid, Mailgun or Postmark but open source and ready for you to run on your own servers.

[Get started now](https://docs.postalserver.io/getting-started)

[Watch a video](https://www.youtube.com/watch?v=d1Lzw_Q_fJQ)

![Image 4](https://docs.postalserver.io/screenshots.png)

Comprehensive Web Interface

Manage your mail servers and view mail logs through an easy to user web interface.

Scalable

Easily horizontally scale your mail service up and down as needed.

Webhooks

Set up webhooks to notify HTTP endpoints when messages are delivered or have issues.

Development Mode

Automatically hold messages in Postal as needed to handle testing during development.

IP Pools

Send mail from different IP addresses to help build and maintain good IP reputation.

Spam & anti-virus integrations

Integrate with SpamAssassin, rspamd and ClamAV to have incoming & outgoing messages checked for threats and spam.

[Discover more features](https://docs.postalserver.io/welcome/feature-list)
