Pines is a library of animations, sliders, tooltips, accordions, modals, and more! It's a set of UI elements that can be copy and pasted into any [Alpine](https://alpinejs.dev/) and [Tailwind](https://tailwindcss.com/) projects.

[View The Elements](https://devdojo.com/pines/docs/accordion)[View on Github](https://github.com/thedevdojo/pines)

What is Pines?

Pines is a UI library built for AlpineJS and TailwindCSS.

How do I install Pines?

Add AlpineJS and TailwindCSS to your page and then copy and paste any of these elements into your project.

Can I use Pines with other libraries or frameworks?

Absolutely! Pines works with any other library or framework. Pines works especially well with the TALL stack.

All you need is [Alpine](https://alpinejs.dev/) and [Tailwind](https://tailwindcss.com/) and you're good to go! 

 Learn why we built it this way, by [diving into the docs](https://devdojo.com/pines/docs).

