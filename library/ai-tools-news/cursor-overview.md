# Cursor Documentation

Cursor is an AI editor and coding agent. Use it to understand your codebase, plan and build features, fix bugs, review changes, and work with the tools you already use.

![Image 1: Welcome to Cursor, the AI editor and coding agent](https://docs.cursor.com/docs-static/_next/image?url=%2Fdocs-static%2Fimages%2Fagent%2Freview.jpg&w=1920&q=75&dpl=dpl_3JmJLbhDDWTxz2re9TL6NHhDBxPn)
