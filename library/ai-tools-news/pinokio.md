# Launch Anything.

Easiest way for humans and AIs to install, launch, and control any app.

[Download](https://pinokio.computer/download.html)[Documentation](https://pinokio.computer/docs)[Newsletter](https://pinokiocomputer.substack.com/)

Available for macOS, Windows, and Linux

01

1-click automation workflows

### App Launcher

Install, start, and control apps, agents or CLI tools with one-click launchers instead of wiring together setup and automation steps by hand.

[Browse Published Launchers](https://beta.pinokio.co/)[Build a Launcher](https://pinokio.computer/docs/#/?id=_1-build-a-1-click-launcher)

![Image 3: Launcher screenshot](https://pinokio.computer/launcher.gif)

02

Agent-App Interpreter

### Agent Interpreter

All apps in Pinokio are instantly agent-ready without you doing anything. Any AI agent can automatically launch and control any app in pinokio, with zero config. Works with all agents (Claude, Openclaw, Codex, Gemini, Cursor, and more)

[How it works](https://pinokio.computer/docs/#/?id=_14-agent-interpreter)

![Image 4: Agent screenshot](https://pinokio.computer/openclaw.png)

03

Local runtime platform

### Localhost Cloud

Run apps, agents, and web services on your own machine with runtimes, package managers, routing, and shell access already built in.

[Localhost Search Engine](https://pinokio.computer/docs/#/?id=_8-lww-lan-wide-web)[Local App Domains](https://pinokio.computer/docs/#/?id=_9-instant-https-web-domains)

![Image 5: Localhost Cloud screenshot](https://pinokio.computer/localhost.png)

04

Contextual app help

### App Assistant

Launch an app-aware agent inside any Pinokio app to ask questions, fix bugs, and even customize the app without leaving the workflow.

[How it works](https://pinokio.computer/docs/#/?id=_12-ask-ai)

![Image 6: App Assistant screenshot](https://pinokio.computer/askai2.gif)

05

Task-focused workspaces

### Agent Launcher

Create task-focused workspaces with custom skills that are guaranteed to trigger, so each agent starts with the right tools, context, and behavior for the job.

[How it works](https://pinokio.computer/docs/#/?id=_15-agent-launcher)

![Image 7: Agent Launcher screenshot](https://pinokio.computer/docs/start_agent_modal.png)
