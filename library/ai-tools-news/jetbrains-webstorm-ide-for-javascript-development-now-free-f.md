## Excerpt

JetBrain’s IDE for JavaScript is now free for non-commercial use. Webstorm thus joins RustRover (Rust IDE) and Aqua (test automation IDE) to the list of JetBrain IDEs available under a non-commercial licensing model. Non-commercial free license users must agree to usage data collection to enjoy the same features as those with paid subscriptions, including AI-powered code completion.
