Video to Website

**How to install the Video to Website skill:**

1.   In your project folder, create the directory `.claude/skills/video-to-website/`

2.   Drop the `SKILL.md` file into that folder.

3.   Open Claude Code in that project. Type `/video-to-website` to confirm it shows up.

4.   Give Claude a video file and say "turn this into a scroll-driven website" — or type `/video-to-website` and describe what you want.

That's it. Claude will extract frames, build a premium scroll-driven animated website with GSAP and canvas rendering, and serve it locally for you to preview.
