## Portable Commands — Cursor commands now work everywhere.

Point Field Theory at any folder of markdown commands. They become available in Claude, ChatGPT, Cursor, or anywhere else you type.

command + shift + k works locally

![Image 1: Portable commands feature showing Cursor commands available everywhere](https://www.fieldtheory.dev/images/portablecommands.gif)

## Priority Mic — Never lose your input to Bluetooth again.

Mac likes to switch your microphone when Bluetooth headphones connect. Field Theory doesn't. Set your preferred input once. It stays selected until you change it.

works locally

![Image 2: Priority Mic feature showing microphone selection dropdown](https://www.fieldtheory.dev/images/priority-mic.png)

## Voice Commands — "Run the deslop command" and it will just run.

While transcribing you can say "use the commit command" and Field Theory will insert it. This works with any command in your directory and without internet. Just invoke what you need mid-sentence.

[See the dedicated Hot Mic page and full default command list.](https://www.fieldtheory.dev/hot-mic.html)

works locally

![Image 3: Verbal commands in Claude Code showing logs, commit, and review commands](https://www.fieldtheory.dev/images/voicecommands.png)

## Draw on screenshots. Paste the result.

Capture a screenshot. Circle the problem. Arrow to where it should go. Freehand a quick wireframe. It's all pasteable without exporting from another tool.

works locally

![Image 4: Drawing feature showing freehand sketching and copy/paste](https://www.fieldtheory.dev/images/draw.gif)

## Speak like you're human. Paste like you proofread.

Auto-improve removes filler, tightens phrasing, and labels your attachments—so spoken prompts read like written ones.

**Before:** "So the auth tokens are expiring, well actually they're refreshing but then failing silently, not sure if it's the token endpoint or maybe client-side storage, anyway I added a screenshot of the error from the network tab"

**After:** "The auth tokens refresh but fail silently. It could be the token endpoint or client-side storage. [Figure 1]"

requires internet

![Image 5: Auto-improve settings showing AI enhancement toggle](https://www.fieldtheory.dev/images/auto-improve.png)

## Input field agnostic — Works anywhere you can paste.

No integrations to configure. Field Theory uses the clipboard. ChatGPT, Claude, Cursor, Copilot, Gemini, Perplexity—if it accepts paste, it works.
