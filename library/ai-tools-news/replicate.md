### Deploy custom models

You aren’t limited to the models on Replicate: you can deploy your own custom models using[Cog](https://github.com/replicate/cog), our open-source tool for packaging machine learning models.

Cog takes care of generating an API server and deploying it on a big cluster in the cloud. We scale up and down to handle demand, and you only pay for the compute that you use.

[Learn more](https://replicate.com/docs/guides/deploy-a-custom-model)
