## A0 Connector

### Install A0 CLI connector and start working on your computer.

Agent Zero runs in a sandboxed environment. A0 CLI opens the door to your own machine.

Requires **Agent Zero 1.9 or newer**. Earlier versions do not support the CLI connector.

1.   **Install A0 CLI with one command.** That's it.  Install A0 CLI for macOS / Linux Windows  # One command. Then Agent Zero can work on this machine.$`curl -LsSf https://cli.agent-zero.ai/install.sh | sh`  # One command. Then Agent Zero can work on this machine.>`irm https://cli.agent-zero.ai/install.ps1 | iex`     
2.   **Launch `a0` in your terminal.**
3.   **Start working on real files.** Agent Zero is working where you work.

## PLUGINS AND SKILLS

### Extensible with Plugins and Skills

Agent Zero's ecosystem is driven by Plugins. Wrap your skills, tools, and custom behaviors into shareable plugins that anyone can install.

From simple tools to complex workflows, plugins encapsulate expertise. They can extend the agent's capabilities instantly with new integrations, prompts, and behaviors.

### a0-development

#### Framework Development Skill

The complete guide to extending Agent Zero. Architecture, tools, extensions, API endpoints, agent profiles, prompts, and the plugin system.

Create new Tools & Extensions

Build API Endpoints

Design Agent Profiles

Understand the Agent Loop

Master the Prompt System

#### Plugin System Skills

### a0-plugin-router

#### Plugin Router

The intelligent dispatcher. Describe what you need and it routes to the right specialist skill automatically.

### a0-create-plugin

#### Create Plugin

Full-stack plugin development: manifest, tools, extensions, frontend UI, settings.

### a0-review-plugin

#### Review Plugin

4-phase audit covering manifest validation, structure, code patterns, and security with PASS/FAIL/WARN verdicts.

### a0-manage-plugin

#### Manage Plugins

Browse the Plugin Hub, scan for security, install, update, enable, disable, or uninstall via API or UI.

### a0-contribute-plugin

#### Contribute Plugin

Publish to the Plugin Index. Repo setup, index.yaml, CI validation, and PR submission step by step.

### a0-debug-plugin

#### Debug Plugin

Plugin not loading? Crashing? Missing? Systematic diagnosis from discovery to frontend rendering.
