# Build real world software with AI.

An open source, terminal-based AI coding agent designed to handle large projects, large files, and large tasks. Featuring a diff review sandbox, full auto mode, and smart context management up to 2M tokens.

[Learn More on GitHub](https://github.com/plandex-ai/plandex)

## Get to **production-ready** with LLMs.

Plandex helps you use generative AI productively in serious projects.

Build entire features and apps with a robust, dev-friendly workflow.

### Configurable Autonomy

Go from full auto mode to granular step-by-step control. Plandex adapts to your task and comfort level.

### Keep Changes Isolated

Stage and review diffs across many files. Execute commands, roll back on failure, and automatically debug.

### Combine The Best Models

Plandex mixes models from Anthropic, OpenAI, Google, and more. Avoid lock-in and use the right model for each stage.

### Work With Large Projects And Files

With a 2M token effective context window and tree-sitter project maps, Plandex handles heavier tasks than other tools.
