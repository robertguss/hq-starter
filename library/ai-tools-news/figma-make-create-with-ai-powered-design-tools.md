## Your playground for new ideas

Explore every big, wild, wacky, and teeny-tiny idea to find the best ones to run with. Create high-fidelity prototypes to help everyone see your vision.

![Image 20: a prompt box covered by a library modal with foundations and a ui library](blob:http://localhost/80c27d5440e01168e40cd546dfe549a8)![Image 21: a prompt box covered by a library modal with foundations and a ui library](https://cdn.sanity.io/images/599r6htc/regionalized/1355fa0009e2c0d20d99c594fb4920d4ada6c89e-1560x1248.png?w=780&h=624&q=75&fit=max&auto=format)

### Keep everything looking and feeling like your product

Add styling context from your Figma library to stay visually consistent with your design system. From there, you can set custom rules or paste in a frame to guide Figma Make as it brings your idea to life.

[Learn more](https://help.figma.com/hc/en-us/articles/33024539096471-Bring-style-context-from-a-Figma-Design-library-into-Figma-Make)

### Prompt your way to a polished design

Select the part of the design you want to continue refining and prompt AI to change the appearance. Even if you aren’t sure how to design or code the outcomes yourself, AI can help you get there.

[Learn more](https://help.figma.com/hc/en-us/articles/31304485164695)

![Image 24: Design interface showing content categorization with "All", "Biology", and "Ecology" filter tags, along with typography controls and a font selection dropdown featuring Hepta Slab, Instrument Sans, Instrument Serif, and Inter typefaces.](blob:http://localhost/45671d187a256d18a59d53ae6c32dd21)![Image 25: Design interface showing content categorization with "All", "Biology", and "Ecology" filter tags, along with typography controls and a font selection dropdown featuring Hepta Slab, Instrument Sans, Instrument Serif, and Inter typefaces.](https://cdn.sanity.io/images/599r6htc/regionalized/6239a4bd10ac08eb101cc4b628c3944f5ae9999a-1080x864.png?w=1080&h=864&q=75&fit=max&auto=format)

### Edit and fine-tune AI outputs

Collaborate with AI. You can directly edit what Figma makes—whether that means rewriting copy, replacing images, or changing padding and margins. Or, copy your preview as design layers to continue iterating in Figma Design.

[Learn more](https://help.figma.com/hc/en-us/articles/31304485164695)

![Image 26: a figma make icon next to a supabase icon with a prompt box under](blob:http://localhost/8c4185f522dc1c8e5bbbbc0ce282bbd4)![Image 27: a figma make icon next to a supabase icon with a prompt box under](https://cdn.sanity.io/images/599r6htc/regionalized/e027a2b862f7b9135a64a669b09134a809e0e455-1560x1248.png?w=780&h=624&q=75&fit=max&auto=format)

### Build apps with real data

Connect Figma Make to Supabase to turn your idea into a web app that’s ready to ship—no coding required. Access everything you need to add user authentication, store user data, connect private APIs, and more.

[Learn more](https://help.figma.com/hc/en-us/articles/32640822050199-Add-a-backend-to-a-functional-prototype-or-web-app)

### Figma Make works seamlessly with other Figma products

Bring the power of Figma Make into Figma Sites to easily customize your website, test interactions, and publish. Select any frame in Figma Sites and prompt it to life with Figma Make.

[Learn more](https://help.figma.com/hc/en-us/articles/31242824165143-Guide-to-code-layers-in-Figma-Sites)
