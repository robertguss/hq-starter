# Craft&run scoped Hermes Agents locally.

**Ankh.md** helps you use multiple Hermes Agents on one computer, scoped to individual folders, projects, or specialized use cases.

Each Ankh Hermes Agent gets its own skills, tools, instructions, and [other native Hermes Agent features](https://hermes-agent.nousresearch.com/docs/) using a simple passthrough .yaml config.

Feature Highlights

## The origin for a future of **infinite-π abundance**.

**Ankh.md** was created to help you get things done faster. Hermes Agent brings local AI to your computer, the Ankh makes it scoped & multi-agentic.

Per-Folder Agents

Each folder on your computer gets its own agent. Config, skills, sessions, and memories live in a local .agent folder. Customize each agent's instructions, skills, tools, model, and more.

One Command, Context-Aware

Run hermes anywhere. In an Ankh folder you get the scoped agent; outside you get default Hermes. No manual switching.

Project-Local Skills

Each Ankh Hermes Agent gets its own skill set. For example, a Docs Explorer can navigate & expand your docs, a Web Researcher searches and summarizes info, different skills per project.

Agent Identities

Give each Ankh Hermes Agent a title, expertise, and instructions via .agent/agent.jsonc. Each agent can have its own writing style & identity rules.

Config Passthroughs

Global ~/.hermes baseline plus local .agent/config.yaml overrides. Different models, toolsets, and settings per project.

Scoped Sessions & Memories

Each agent keeps its own conversation history and memories in its .agent folder.

Non-Intrussive Modding

Ankh.md creates a modded version of Hermes Agent, but your default one stays intact & always usable.

Per-Agent API Keys

Generate up-to-date docs, PR summaries, and test cases without lifting a finger.

Native Hermes Features

Web, terminal, file, browser, vision, MCP... all your favorite Hermes Agent tools. Enable or disable toolsets per project in config.

Sub-Agent Delegation

Not officially supported yet, but you can create multiple Ankh Hermes Agents and instruct them to delegate to each other.

Done-for-You Examples

Docs Explorer, Web Researcher, Diagram Maker, Plan Writer, ASCII Designer. Copy any folder and run `hermes` to start.

Git-Friendly Setup

Commit .agent/ to share config, skills, and memories with your team. Ignore memories/ in .gitignore if you want them private.
