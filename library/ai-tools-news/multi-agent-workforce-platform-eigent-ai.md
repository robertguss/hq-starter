## How it works

Eigent open-source cowork work with build with you.

[Get Started](https://www.eigent.ai/download)

### Smart Task Routing & Execution

*   Break complex work into clear, manageable steps
*   Automatically assign each step to the most suitable worker
*   Run tasks on autopilot from start to finish
*   Reassign work intelligently when a task needs a new approach

![Image 42: memory](https://www.eigent.ai/_next/image?url=https%3A%2F%2Fcdn.eigent.ai%2Ffeature%2Fmemory.png&w=828&q=75)

#### Memory

Keep every task, file, and context organised under one project, so your workforce stays focused on the work that matters.

![Image 43: skills](https://www.eigent.ai/_next/image?url=https%3A%2F%2Fcdn.eigent.ai%2Ffeature%2Fskills.png&w=828&q=75)

#### Skills

Use skills with Eigent to turn your knowledge into actions.

![Image 44: triggers](https://www.eigent.ai/_next/image?url=https%3A%2F%2Fcdn.eigent.ai%2Ffeature%2Ftriggers.png&w=828&q=75)

#### Trigger

Schedule runs, set condition-based actions, and make sure work happens at the right time.

![Image 45: worker](https://www.eigent.ai/_next/image?url=https%3A%2F%2Fcdn.eigent.ai%2Ffeature%2Fworker.png&w=828&q=75)

#### Worker

Create as many workers as you need, customised with your own tools, skills, and workflows.

Build with control and integration in mind

### Powerful Dashboard Controls across all projects

*   Organise projects and tasks in one place
*   Manage agent settings, models, skills, and memory
*   Connect your workforce to external channels (coming soon)
*   Extend capabilities with tools and MCPs
*   Work with browser control for real-world tasks, human-in-the-loop review, and sensitive actions that stay under your control

![Image 46: dashboard](https://www.eigent.ai/_next/image?url=https%3A%2F%2Fcdn.eigent.ai%2Ffeature%2Fdashboard.png&w=1200&q=75)

### Bring Your Own Keys or Models

Use Eigent Cloud, your own API key, or local models with flexible switching and optimised setup options.

![Image 47: models](https://www.eigent.ai/_next/image?url=https%3A%2F%2Fcdn.eigent.ai%2Ffeature%2Fmodel.png&w=1200&q=75)

Local, Private and Secure by Design

Choose how you run your agents with your own API keys or local open-source models.

#### Data Governance
