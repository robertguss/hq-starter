#### What is this site?

Learning How to Learn (LHTL) is a positive feedback loop (exponential growth). The faster you learn how to **learn**, the faster you **learn**how to learn.

~8/9 years ago I started focusing on learning how to learn once I saw it was a positive feedback loop, and only within the last few years have I seen meaningful compounding effects. This site, LHTL.org, contains articles on learning concepts/strategies to speed up your own LHTL loop.
